# My Blog Website

A modern, responsive blog website built with HTML, CSS, and JavaScript.

## Features

- 📱 Fully responsive design
- ✨ Smooth navigation and scrolling
- 📝 Dynamic blog posts
- 💌 Contact form
- 🎨 Beautiful gradient design
- 🚀 Lightweight and fast

## Project Structure

```
my--first-website/
├── index.html      # Main HTML file
├── style.css       # Styling
├── script.js       # JavaScript functionality
└── README.md       # Project documentation
```

## Sections

1. **Navigation** - Easy navigation between sections
2. **Hero Section** - Eye-catching introduction
3. **Blog Posts** - Display blog posts with hover effects
4. **About** - About me section
5. **Contact** - Contact form
6. **Footer** - Footer with copyright

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/hudasifat2-coder/my--first-website.git
   ```

2. Open `index.html` in your web browser to view the blog.

## Customization

### Add More Blog Posts
Edit the `blogPosts` array in `script.js`:

```javascript
const blogPosts = [
    {
        id: 1,
        title: "Your Post Title",
        date: "Date",
        excerpt: "Post excerpt",
        content: "Full content"
    },
    // Add more posts...
];
```

### Change Colors
Edit the color values in `style.css` to match your style.

### Update Content
Edit the HTML in `index.html` to personalize your blog.
